Basic Data Access

Current Item Data

Simple Field Access
Get a field from the current item - the most basic n8n operation
{{ $json.fieldName }}
Fields with Spaces
Access field names containing spaces or special characters
{{ $json["First Name"] }}
{{ $json["Email Address"] }}
{{ $json["Phone Number"] }}
Nested Object Access
Access nested properties in complex objects
{{ $json.user.email }}
{{ $json.customer.address.city }}
{{ $json.order.items[0].name }}
Safe Array Access
Safely access array items with fallbacks
{{ $json.items?.[0] }}              // First item (safe)
{{ $json.items?.[-1] }}             // Last item (safe)
{{ $json.items?.length || 0 }}      // Array length

Data from Other Nodes

Specific Node Data
Get data from a specific node in your workflow
{{ $('HTTP Request').item.json.id }}
{{ $('Set Data').item.json.userId }}
{{ $('Code').item.json.result }}
All Items from Node
Work with multiple items from a single node
{{ $('Get Users').all() }}           // All items as array
{{ $('Get Users').all().length }}    // Count items
{{ $('Get Users').first().json.name }} // First item only

Date & Time Functions

💡 Pro Tip: n8n uses Luxon for date handling

All date functions are powered by Luxon, giving you powerful date manipulation capabilities.

Current Date & Time

Basic Date Functions
Get current date and time in various formats
{{ $now }}                          // Current timestamp
{{ $today }}                        // Today at midnight
{{ $now.format('YYYY-MM-DD') }}     // 2024-01-15
Date Calculations
Add or subtract time from dates
{{ $now.plus({days: 7}) }}          // 7 days from now
{{ $now.minus({hours: 2}) }}        // 2 hours ago
{{ $now.startOf('day') }}           // Beginning of today

String Manipulation

Text Transformations

Case Conversion
Change text case and capitalization
{{ $json.name.toLowerCase() }}     // lowercase
{{ $json.name.toUpperCase() }}     // UPPERCASE
{{ $json.name.trim() }}            // Remove spaces
String Extraction
Extract parts of strings
{{ $json.text.slice(0, 50) }}      // First 50 characters
{{ $json.email.split('@')[0] }}     // Username from email
{{ $json.text.length }}             // String length

Array Processing

Array Operations

Basic Array Access
Access and count array items
{{ $json.items.length }}           // Count items
{{ $json.items[0] }}                // First item
{{ $json.items[-1] }}               // Last item
Array Filtering
Filter and search arrays
{{ $json.items.filter(item => item.active) }}
{{ $json.items.find(item => item.id === '123') }}
{{ $json.items.map(item => item.name) }}
